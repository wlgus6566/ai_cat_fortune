# 🔮 사주 기반 고민 상담 AI 챗봇 UI/UX 디자인 가이드

---

## 1. 🎨 디자인 컨셉 및 원칙

### 1.1 디자인 철학

- **마법 같은 상담 경험**  
  운세를 마치 작은 동화 속 모험처럼 느낄 수 있도록, 마법사 고양이와 감성적인 일러스트 중심으로 구성합니다.

- **정서적 안정과 친밀감**  
  따뜻한 색감과 귀여운 캐릭터로 사용자가 편안하게 접근할 수 있는 상담 환경을 제공합니다.

- **현대적 감성과 전통적 상징의 조화**  
  별, 달, 오행 등 전통 운세 상징을 현대적인 UI 속에 자연스럽게 녹입니다.

---

### 1.2 사용자 경험 목표

- 감성적 몰입 유도 (일관된 감성 디자인 유지)
- 개인 맞춤화 경험 강화 (사주 기반 콘텐츠 제공)
- 탐색이 쉬운 구조 (단순한 탭 네비게이션)

---

### 1.3 접근성 고려사항

- 충분한 색상 대비로 정보 전달의 명확성 확보
- 텍스트 중심 기능에 적절한 라벨링 및 시각 보조 요소 제공
- 모바일 환경에서 터치 영역 충분히 확보

---

## 2. 🧭 사용자 흐름 및 와이어프레임

### 2.1 사용자 흐름도

```mermaid
graph TD
A[로그인/회원가입] --> B[프로필 설정]
B --> C[오늘의 운세 확인]
C --> D[고민 상담 (챗봇)]
D --> E[프로필 편집/기록 보기]
```

---

### 2.2 와이어프레임 개요 (텍스트 기반)

#### 1) 로그인 화면

- **배경**: 마법의 창문 앞 밤하늘 + 고양이 마법사 일러스트
- **중앙**: 캐릭터 + 책 → 별 반짝임 애니메이션
- **버튼**:
  - [Google Login] (화이트)
  - [Kakao Login] (연노랑)

---

#### 2) 오늘의 운세 - 보기 전 화면

- **상단바**:

  - 타이틀: 오늘의 운세
  - 우측: 설정 아이콘 (⚙️)

- **메인 콘텐츠**:

  - 🐱 마법사 고양이 캐릭터 (애니메이션 포함)
  - 💬 말풍선: "오늘 운세를 점쳐볼까냥~?"
  - 🔮 [오늘의 운세 보기] 버튼
    - 보라색 pill형 / glow + 흔들림 효과
  - 📝 사주 키워드 예시: "당신의 오늘 기운은 [수(水)] 계열이에요"

- **하단 네비게이션 바**:
  - [오늘의 운세] [상담하기] [프로필]

---

#### 3) 오늘의 운세 - 결과 화면

- 🐱 고양이 캐릭터 + 말풍선

  - "오늘 하루의 기운을 점쳐봤다냥~"

- 🔮 오늘의 종합 운세

  - 점수 시각화: ○●●●● 형식 or 원형 차트
  - 키워드 예시: "조심 / 설렘 / 균형"
  - 종합 해석:
    ```
    오늘은 생각보다 변수가 많은 하루예요.
    특히 인간관계에선 의도치 않은 오해가 생길 수 있으니,
    말 한마디에 신중하세요.
    ```

- 📊 카테고리별 운세 카드

  - 💖 연애운 | 💰 금전운 | 🧘 건강운 등
  - 점수 + [더보기 ▶] → 팝업

- [AI 프로필 생성하기] 버튼
  - 사용자의 오행 기반 캐릭터 생성 기능

---

#### 4) 챗봇 화면

- **상단**: AI 고양이 챗봇 (사주 스타일)
- **중앙**: 채팅 UI
  - 사용자: 오른쪽, 흰 배경
  - AI 응답: 왼쪽, 연베이지톤 + 감성 이모지
- **하단**: 채팅 입력창 + 전송 버튼
  - 부적 이미지 제공 시: 클릭 → 확대/저장

---

#### 5) 프로필 및 설정

- AI 생성 아바타 (오행 반영)
- 사용자 정보: 이름, 성별, 생년월일, 태어난 시간
- 편집 버튼 포함

## 3. 📐 화면 구성 및 와이어프레임

### 1) 궁합 입력 화면

- **상단 타이틀**: `궁합 보기`
- **설명 텍스트**:
  `"고양이 점술사가 사랑의 궁합을 봐줄게냥~ 😺💫"`
- **입력 카드** 2개 (사용자 / 상대):
  - 이름
  - 성별 (선택 버튼)
  - 생년월일 (달력 입력 또는 텍스트)
  - 태어난 시간 (Drop-down 또는 직접 입력)
- **[궁합 점쳐보기] 버튼**:
  - **스타일**: 보라색 glow + 흔들림 애니메이션
  - **클릭 시**: 별이 도는 형태의 로딩 애니메이션

---

### 2) 궁합 결과 화면

- **상단 요소**:
  - ✨ **궁합 결과 제목** (예: `"별빛 아래 운명의 실타래"`)
  - `"오늘의 궁합 테마는 [궁합 테마]야~!"`
- **요약 카드**:
  - **점수**: 원형 차트 (0~100)
  - **한 줄 요약**: `"지금 이 순간, 둘은 꽤 잘 어울리는 인연이야!"`
- **궁합 세부 카드 (아코디언 or 섹션 분할)**:
  | 항목 | 내용 예시 |
  | ------------- | ------------------------------------------------------- |
  | 성격궁합 | 점수 + 감성 분석 + 고양이 조언 |
  | 연애스타일 | 점수 + 분석 + 짧은 팁 |
  | 갈등요소 | 점수 + 분석 + 예방 팁 |
  | 미래전망 | 점수 + 장기 전망 + 조언 |
  | 음양오행 분석 | 두 사람의 오행/음양 + 상생/상극 관계 설명 + 고양이 해석 |
- **하단 콘텐츠**:
  - 🧙‍♂️ **전체 조언**: `"마음의 속삭임을 잘 들어주는 것만으로도, 사랑은 자라난다냥~"`
  - 🐾 **고양이의 한마디**: `"둘이 닮은 듯 다른 게 매력이라냥!"`
  - 🎁 **행운 아이템**: `"달 모양 목걸이"`
  - 💑 **추천 데이트**: `"별이 잘 보이는 공원 산책"`
