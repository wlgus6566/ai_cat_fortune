## 1. 🎨 디자인 컨셉 및 원칙

### 1.1 디자인 철학

- **마법 같은 상담 경험**: 운세를 마치 작은 동화 속 모험처럼 느낄 수 있도록 마법사 고양이와 감성적인 일러스트를 중심으로 구성합니다.
- **정서적 안정과 친밀감**: 따뜻한 색감과 귀여운 캐릭터를 통해 사용자가 편안하게 접근할 수 있는 상담 환경을 제공합니다.
- **현대적 감성과 전통적 상징의 조화**: 별, 달, 오행 등 전통 운세의 상징을 현대적인 UI 속에 자연스럽게 녹입니다.

### 1.2 사용자 경험 목표

- **감성적 몰입 유도**: 로그인부터 챗봇 상담까지 일관된 분위기의 감성 디자인 유지
- **개인 맞춤화 경험 강화**: 사주 기반 프로필, 부적, 운세 카드 등 사용자 정보에 따라 콘텐츠가 다르게 제공
- **탐색이 쉬운 구조**: 하단 탭 네비게이션 중심의 단순한 플로우 제공

### 1.3 접근성 고려사항

- **충분한 색상 대비**로 정보전달의 명확성 확보
- **텍스트 중심 기능**에 적절한 라벨링 및 시각 보조 요소 제공
- **터치 영역** 충분히 확보 (모바일 최적화)

### 2.2 와이어프레임 개요 (텍스트 표현)

### 1) 로그인 화면

```
[배경] 마법의 창문 앞 고양이 마법사 일러스트 (밤하늘 + 책)
[중앙] 캐릭터 + 책 → 애니메이션 효과로 별이 반짝임
[버튼]
- [Google Login] (화이트)
- [Kakao Login] (연노랑)

```

### 2) 오늘의 운세

```
[상단] “오늘의 운세를 볼꺼냥?” (고양이 캐릭터 말풍선)
[중앙] 운세 보기 버튼 (반짝 효과)

> 운세 본 후:
- [운세 점수 시각화] (원형 차트)
- [카테고리 결과 카드] (연애 / 금전 / 건강 / 인간관계)
- [추가 기능 버튼] AI 프로필 생성하기

```

### 3) 챗봇 화면

```
[상단] AI 고양이 챗봇 프로필 (사용자 사주 반영된 스타일)
[중앙] 채팅 대화창
- 사용자 말풍선 (오른쪽, 흰색 배경)
- AI 응답 (왼쪽, 따뜻한 베이지톤 말풍선 + 이모티콘)

[하단] 사용자 입력 필드 + 전송 버튼

> 부적 제공 시:
- [이미지 카드] → 클릭 시 확대 및 다운로드 기능

```

### 4) 프로필 및 설정

```
[상단] AI 생성 아바타 (오행 기반 스타일링)

## 3. 🧩 UI 컴포넌트 가이드

### 3.1 공통 UI 요소

| 컴포넌트 | 설명 |
| --- | --- |
| 버튼 | 둥근 모서리, 마법 느낌의 그라데이션 효과 (기본: 보라/연보라) |
| 입력 필드 | 말풍선 스타일, 약간 둥근 사각형, 내부 그림자 효과 |
| 카드 UI | 운세 결과용. 부드러운 그림자 + 아기자기한 아이콘 포함 |
| 캐릭터 | 상황에 따라 표정 변화, 사주 기반 커스터마이징 가능 |
| 아이콘 | 별, 달, 북두칠성, 오행 심볼 등 감성적인 픽토그램 활용 |

### 3.2 상태별 디자인 예시

- **버튼**
    - 기본: #990dfa (보라)
    - 호버: #7b08d1
    - 비활성화: #cccccc
- **입력 필드**
    - 기본: #F1F1F5
    - 포커스: 보라색 테두리 + 살짝 빛나는 효과
- **챗 메시지**
    - 사용자: 말풍선 우측, 흰 배경 + 보라색 텍스트
    - AI: 말풍선 좌측, 연베이지 배경 + 귀여운 이모지 삽입 가능

## 4. ✨ 인터랙션 및 애니메이션 지침

### 4.1 인터랙션 설계

- 챗봇 응답 시: "점점점" 타이핑 애니메이션 표시
- 운세 카드 클릭 시: 확대 + 해석 팝업
- 부적 이미지: 펼쳐지는 스크롤 애니메이션 + 다운로드 버튼 제공

### 4.2 애니메이션 효과

- **홈 화면 별 반짝임 효과**: CSS 루프 애니메이션 (부드럽고 은은하게)
- **운세 보기 버튼 클릭 시**: 카드가 공중에 뜨며 펼쳐지는 듯한 애니메이션
- **캐릭터 애니메이션**: 감정 변화에 따른 표정 및 모션 변경

## 5. 🌈 비주얼 스타일 가이드

### 5.1 색상 팔레트

| 용도 | 색상 | 설명 |
| --- | --- | --- |
| 주요 색상 | `#990dfa` | 브랜드 보라색 (마법/신비감 상징) |
| 보조 색상 | `#3B2E7E` | 깊은 신뢰감, 고급스러운 톤 |
| 배경 색상 | `#F9F9F9` | 부드러운 전체 분위기 유지 |
| 포인트 색상 | `#FFD966` | 노란색 포인트 (Kakao 버튼, 하이라이트) |
| 상태 색상 | 성공: #4CAF50 / 경고: #FFC107 / 오류: #F44336 |  |

### 5.2 타이포그래피

- **폰트**: Noto Sans KR + 감성적 헤드라인용 서브 폰트 (예: "나눔 손글씨")
- **텍스트 계층 구조**
    - 타이틀: 20px Bold
    - 부제목: 16px Semi-Bold
    - 본문: 14px Regular
    - 설명/라벨: 12px Light

### 5.3 아이콘 및 이미지 스타일

- **아이콘 세트**: 미니멀+일러스트 느낌의 아이콘, 테두리 없는 플랫 스타일
- **캐릭터/배경 이미지**: 밤하늘, 별, 구름, 책, 달 등을 일러스트화
- **부적 이미지**: 한지 느낌의 종이 질감 표현 + 손글씨 스타일 글귀

#### 3. 🔮 오늘의 종합 운세 요약

##### 점수 시각화

- `"조심 / 설렘 / 균형"` 형식
- 시각적으로 구분되도록 pill 스타일 배지로 표현

##### 종합 해석 텍스트

```
"오늘은 생각보다 변수가 많은 하루예요.  
특히 인간관계에선 의도치 않은 오해가 생길 수 있으니,  
말 한마디에 신중하세요."
```
